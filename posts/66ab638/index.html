Dash Leaflet是一个封装式的Leaflet，主导了开源Javascript的交互式地图。语法类似其他Dash结构，命名规范遵循React-Leaflet。

安装Dash-leaflet

pip install dash-leaflet

运行Dash-leaflet。通过在浏览器中访问 http://127.0.0.1:8050/ 便可得到Leaflet map。

from dash import Dash
import dash_leaflet as dl

app = Dash()
app.layout = dl.Map(dl.TileLayer(), center=[56,10], zoom=6, style={'height': '50vh'})

if __name__ == '__main__':
    app.run_server()

其他推荐/相关安装包：

dash-extensions：增速Dash发展

dash-mantine-components：在Dash中创建用户接口

这个本身是个Dash APP创建的，用dash-mantine-components, dash-extensions, dash-iconify, 和 [dash-down](https://github.com/emilhe/dash-down)。

所有组件都被记录在dash-leaflet组成 API 参照中

对于这些附加文件在组成上， React-Leaflet API component reference  和 Leaflet API reference 同样是最好的资源

leaflet plugins

自定义组件

一些自定义组件也被开发了

  • DivMarker (a customized marker that uses a DivIcon instead of an Icon)

一般来说，这些为了组件的文件是限制在dash-leaflet API参考和这些源代码中的。像是GeoJSON组件，有专门的教程

属性可变

dash-leaflet是建立在最顶端 React Leaflet 基础上，而React Leaflet是建立在Leaflet上。React Leaflet引进了可变属性和不可变属性的概念。可变属性能够在组件船舰后被改变，然而不可变不能做到炸点。如果你发现自己需要改变一个不可变的属性，你要用相应的不可变属性重新创建组件。

属性标签

属性标记在字符串中通过 [MUTABLE] 标签保证是可变的。一个例子就是GeoJSON中的属性 zoomToBoundsOnClick

  • zoomToBoundsOnClick (boolean; optional): If True, zoom to feature bounds on click. [MUTABLE, DL].

这些特别的实行是收录在 [DL] 标签中。指示这属性创建和特别定制给Dash-leaflet。因此文档可能会不可用在底层库中,折线传单中的行为可能会偏离记录的内容。

组件的唯一性

功能属性

在dash-leaflet中，一些组成属性是功能。在Dash中功能操作不能直接作为属性出现的。但是dash-extensions提供少量的隐式选择。举个例子，我们考虑GeoJSON组成这个安装包。

Javascript变量

在dash-extensions 能够使变量的映射在窗口对象到组成属性。因此，如果我饿们创建一个 .js文件在assets文件夹中并写有一下内容：

window.myNamespace = Object.assign({}, window.myNamespace, {  
    mySubNamespace: {  
        pointToLayer: function(feature, latlng, context) {  
            return L.circleMarker(latlng)  
        }  
    }  
});

myNamespace.mySubNamespace中的pointToLayer功能能够被用来作为组成属性。

import dash_leaflet as dl
from dash_extensions.javascript import Namespace
ns = Namespace("myNamespace", "mySubNamespace")
geojson = dl.GeoJSON(data=data, pointToLayer=ns("pointToLayer"))

为了完整性，例如这是一个完整的app

import random
import dash_leaflet as dl
import dash_leaflet.express as dlx
from dash import Dash
from dash_extensions.javascript import Namespace

# Create some markers.
#创建一些标记
points = [dict(lat=55.5 + random.random(), lon=9.5 + random.random(), value=random.random()) for i in range(100)]
data = dlx.dicts_to_geojson(points)
# Create geojson.
#创建geojson
ns = Namespace("myNamespace", "mySubNamespace")
geojson = dl.GeoJSON(data=data, pointToLayer=ns("pointToLayer"))
# Create the app.
#创建app
app = Dash()
app.layout = dl.Map([dl.TileLayer(), geojson], center=(56, 10), zoom=8, style={'height': '50vh'})

if __name__ == '__main__':
    app.run_server()
