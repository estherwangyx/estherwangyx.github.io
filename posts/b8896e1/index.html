等时线表示从特定点在一定时间内可以到达的地理区域。例如，在步行距离的背景下，等时线是城市规划师等专业人士的有用工具，旨在了解特定区域内的可达性和连通性。

2.构建多边形

networkx库用来创建复杂的网络图，一般用它来进行路径分析和聚类分析等。利用netwarkX返回道路网络内符合距离要求的所有节点。将提取的节点绘制成等时线多边形

import networkx as nx 
from shapely.geometry import Point, Polygon  
import geopandas as gpd 

isochrone_polys = []  #定义等时线对象

for time in isochrone_times:  #循环生成四类等时线
    subgraph = nx.ego_graph(G,                     #返回范围内所有节点   #G为街区数据
                            center_node,          
                            radius=time*60, 
                            distance='travel_time')

    node_points = [Point((data['x'], data['y']))     #生成点
                   for node, data in subgraph.nodes(data=True)]   #sudgraph有向图，其中节点标记为支持和不支持的节点

    polygon = Polygon(gpd.GeoSeries(node_points).unary_union.convex_hull)  #生成所有节点结构中的最小凸多边形
    gpd.GeoDataFrame([polygon], columns = ['geometry']).plot()   #生成几何数据，绘制
    isochrone_polys.append(gpd.GeoSeries([polygon])) #将几何数据添加到等时线对象中
