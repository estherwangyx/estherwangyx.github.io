重装系统后，Postgis安装问题

重装后安装过程中遇到的问题及解决办法

系统从win7升级到win10，虽然我把postsql安装在了D盘，重装系统之后还是能在D盘中找到postsql的文件夹。但是打开Pgadmin4里面没有服务器。

随后查看了服务列表，列表中没有postsql服务。又查看了注册表regedit.exe也没有postsql相关。 所以一怒之下重装了。

在重新安装中出现如下问题

1.安装postgis扩展时，出现了 A certificate verification problem was encountered whilist accessing

解决办法：后来多次试验发现是密码大小写弄错了。。。*** 评论里好多人问，这一步的密码是新建服务器密码，不是登录密码***

2.不能创建spatial database模块，Spatial database creation failed

解决办法：利用管理员身份运行postgis安装程序，解决！

3.不能打开postgis,文件中缺少libintel_9.dll文件

解决办法：找到postgisgui文件夹，复制其中的libintel_8.dll文件更名为libintel_9.dl
