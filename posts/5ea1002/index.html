Arcpy将文本信息转化为shp文件

wangyx 收录于 Python
2020-03-06 约 100 字 预计阅读 1 分钟 

很多Arcgis坐标数据文件为存储在表格的文本形式，需要经过处理才能将文本数据转化为地理数据。下方代码能将多个文本信息批量转化为Arcgis中的图层

import arcpy
import os
from arcpy import env
env.workspace = "E:\\01ArcPy\\CSV"
lys = arcpy.ListFiles()
for ly in lys:
    arcpy.MakeXYEventLayer_management(ly,"经度","纬度",ly)

由于以上代码只是将数据展现在Arcgis中，并没有生成地理数据文件，因此还需要将图层输出为shp格式文件。在循环中最后一行添加下方代码即可

arcpy.FeatureClassToShapefile_conversion(layer,output)

建立Arcgis工具箱，方便以后直接使用。

import sys
