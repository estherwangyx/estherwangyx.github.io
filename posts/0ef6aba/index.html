3.2Pandas对象介绍 

import numpy as np
import pandas as pd

1.Pandas的Series对象

Serries对象为带索引数据构成的数组

data = pd.Series([0.25,0.5,0.75,1.0])
data
0    0.25
1    0.50
2    0.75
3    1.00
dtype: float64

Series对象：索引——数组。可通过数值&索引获取数据

data.values
array([0.25, 0.5 , 0.75, 1.  ])

#通过索引获取数据
data[1]
0.5

data[1:3]
1    0.50
2    0.75
dtype: float64

(1) Series是通用的np数组 

data = pd.Series([0.25,0.5,0.75,1.0],index=['a','b','c','d'])
data
a    0.25
b    0.50
c    0.75
d    1.00
dtype: float64

data['b']
0.5

data = pd.Series([0.25,0.5,0.75,1.0],index=[2,5,3,7])
data
2    0.25
5    0.50
3    0.75
7    1.00
dtype: float64

(2)Series是特殊的字典 

#创建Series对象，让对象与字典对应更清晰
population_dict = {
    'Califonia':38332521,
    'Texas':26448193,
    'New York':19651127,
    'Florida':19552860,
    'Illinoios':12882135
}
popuation = pd.Series(population_dict)
popuation
Califonia    38332521
Texas        26448193
New York     19651127
Florida      19552860
Illinoios    12882135
dtype: int64

popuation['Califonia']
38332521

#数组形式
popuation['Califonia':'Illinoios']
Califonia    38332521
Texas        26448193
New York     19651127
Florida      19552860
Illinoios    12882135
dtype: int64
